5. GPU (Graphics Processing Unit) 

What is a GPU?

GPU is responsible for processing graphics and displaying images on the monitor.

Uses:

* Gaming
* Video Editing
* 3D Modeling
* Artificial Intelligence
* Machine Learning

Types of GPU:

Integrated GPU

Built inside the CPU.

Example:

Intel UHD Graphics

Dedicated GPU

Separate graphics card.

Examples:

* NVIDIA GeForce
* AMD Radeon

Analogy

CPU = General Manager

GPU = Graphics Specialist

6. Storage Devices 

Storage devices save data permanently.

HDD (Hard Disk Drive)

Features:

* Uses spinning magnetic disks.
* Larger storage capacity.
* Lower cost.

Advantages:

* Affordable
* High capacity

Disadvantages:

* Slower speed
* Mechanical parts can wear out

---

## SSD (Solid State Drive)

Features:

* No moving parts.
* Uses flash memory.
* Faster performance.

Advantages:

* Fast boot time
* Faster file access
* Durable

Disadvantages:

* More expensive than HDD

---

## HDD vs SSD

| Feature      | HDD     | SSD            |
| ------------ | ------- | -------------- |
| Speed        | Slow    | Fast           |
| Moving Parts | Yes     | No             |
| Durability   | Lower   | Higher         |
| Cost         | Cheaper | More Expensive |
| Boot Time    | Longer  | Shorter        |

---

# How All Components Work Together 

Example: Opening Microsoft Word

1. User clicks Word icon.
2. SSD/HDD sends program data.
3. RAM temporarily stores active data.
4. CPU processes instructions.
5. GPU displays the application.
6. Motherboard connects everything.

### Flow

Storage → RAM → CPU → GPU → Monitor

### Creating a Directory Inside Another Directory and Writing to a File (Windows Command Prompt)

#### 1. Open Command Prompt

* Press **Win + R**.
* Type **cmd**.
* Press **Enter**.

#### 2. Navigate to the Location Where You Want to Create the Folder

For example, to go to your Desktop:

```cmd
cd %USERPROFILE%\Desktop
```

#### 3. Create the Main Directory

```cmd
mkdir Project
```

#### 4. Create a Subdirectory Inside the Main Directory

```cmd
mkdir Project\Docs
```

#### 5. Navigate to the Subdirectory

```cmd
cd Project\Docs
```

#### 6. Create a File

```cmd
type nul > notes.txt
```

Alternatively:

```cmd
echo. > notes.txt
```

#### 7. Write Text to the File

```cmd
echo Hello World > notes.txt
```

#### 8. Append Additional Text to the File

```cmd
echo This is the second line >> notes.txt
```

#### 9. View the File Contents

```cmd
type notes.txt
```

**Output:**

```text
Hello World
This is the second line
```

### Complete Example

```cmd
cd %USERPROFILE%\Desktop
mkdir Project
mkdir Project\Docs
echo Hello World > Project\Docs\notes.txt
type Project\Docs\notes.txt
```
